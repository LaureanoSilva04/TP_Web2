## Temática del TPE
**Turismo Internacional**

## Descripción de la temática
Este sitio web es una plataforma de gestión turística diseñada para administrar información sobre diferentes países y sus ciudades destacadas. Utilizando una relación de 1 a N (País - Ciudad), el sistema permite organizar destinos turísticos y sus características.
